# Postdoc Opportunities

> **Index key format:** `P001`, `P002`, … — used across all three tables to link records.
> **Status tags:** `[OPEN]` `[CLOSED]` `[APPLIED]` `[INTERVIEW]` `[OFFER]` `[REJECTED]` `[DECLINED]`

**Column notes:**

**Column notes:**
- **Portal URL** — URL of the application submission portal (may differ from the job posting link in Table A)
- **CV / Cover Letter / Transcripts / Research Statement / Writing Sample / Teaching Statement / Diversity Statement** — `Y` / `N` / `Optional`
- **# Rec Letters** — integer (e.g., `3`); see `RECOMMENDERS.md` for per-recommender status
- **Reference Code** — application portal job ID or internal code, if any
- **Materials Ready** — brief checklist of prepared items, e.g., `CV✓ CL✓ RS pending` or a completion date

---

## How to Add a New Position

1. Assign the next `P###` ID.
2. Fill one row in **Table A** and one row in **Table B** with the same ID.
3. Add a matching row to `PROGRESS.md` (Tables C & D).
4. Set status to `[OPEN]` until you submit; update as it changes.
