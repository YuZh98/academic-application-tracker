# Seed templates

Original hand-maintained markdown templates from the project's pre-app
era (early Phase 0/1), kept here as a reference shape for future
schema or export-format revisions. Phase 6 superseded them with the
live `exports/OPPORTUNITIES.md` / `PROGRESS.md` / `RECOMMENDERS.md`
generated by `exports.py` after every database write — so these are
read-only design artifacts, not the source of truth.

If you're looking at the live tracker output, see the `exports/`
directory at the repo root (gitignored locally; regenerated on every
DB write). If you're looking for the page contracts these templates
inspired, see `DESIGN.md` §7 and §8.
