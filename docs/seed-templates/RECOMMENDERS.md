**Column notes:**
- **Relationship** — e.g., `PhD Advisor` / `Committee Member` / `Collaborator` / `Postdoc Supervisor`
- **Asked Date** — date you sent the request (`YYYY-MM-DD`)
- **Confirmed** — `Y` once recommender agreed; `N` if declined; blank if pending
- **Submitted Date** — date the letter was submitted to the portal
- **Reminder Sent** — date of last follow-up if needed; leave blank if they submitted on time

## How to Use

1. When you decide to apply to a position, add one row per required recommender.
2. Send the request → fill **Asked Date**.
3. Once they confirm → mark **Confirmed = Y**.
4. If no submission after 1–2 weeks, send a reminder → fill **Reminder Sent**.
5. Once submitted → fill **Submitted Date** and update Table C in `PROGRESS.md` (`All Recs Submitted = Y`) if everyone for that position is done.
